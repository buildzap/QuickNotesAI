English Voice Commands Guide
Quick Start
Click the microphone button to start voice recording
Speak in English - use natural language commands
Say "Submit" or "Save" to automatically create the task
Enhanced: Title and Description fields support long voice input with auto-expansion
Field Commands
"Set title to [task name]"
Set the task title. Example: "Set title to team meeting"
"Set description to [details]"
Set the task description. Example: "Set description to prepare agenda for weekly meeting"
"Set priority to [high/medium/low]"
Set task priority level. Example: "Set priority to high"
"Set status to [pending/in-progress/completed]"
Set task status. Example: "Set status to in-progress"
"Set due date to [date/time]"
Set due date. Example: "Set due date to tomorrow" or "Set due date to next Friday"
"Set tags to [tag1, tag2]"
Set task tags. Example: "Set tags to meeting, work, urgent"
Action Commands
"Submit task" or "Save task"
Create and save the current task
"Add new task"
Clear form and start a new task
"Update task"
Save changes when editing an existing task
"Clear" or "Reset"
Clear all form fields
Example Commands
"Set title to team meeting, set description to prepare agenda for weekly team meeting, set priority to high, set tags to meeting, work, submit"
"Set title project review, set status in-progress, set due date next week, save task"
"Set priority high, set status completed, update task"
Pro Tip: Combine multiple commands in one sentence for faster task creation and editing
Tips for Better Recognition
Speak clearly and at a normal pace
Use a quiet environment for better accuracy
You can edit fields manually after voice input
Input Method automatically changes to "Voice" when using microphone
When editing a task, use "Update Task" voice command to save changes
